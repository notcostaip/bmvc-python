Sobre o projeto

Arquitetura BMVC
em Python puro

Este projeto demonstra a implementacao do padrao Model-View-Controller utilizando exclusivamente a biblioteca padrao do Python 3, sem frameworks externos. O servidor HTTP, o roteamento, a API REST e a camada de dados sao todos construidos do zero, evidenciando os principios de Orientacao a Objetos como encapsulamento, getters e setters.

Python 3 BMVC OO http.server API REST HTML / CSS / JS Zero dependencias
Model — produto.py
Dados, atributos privados, getters e setters
View — index.html
Interface visual com CSS e JS externos
Controller — produto_controller.py
Logica, roteamento e API REST
Server — main.py
